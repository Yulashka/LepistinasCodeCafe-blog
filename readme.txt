== Description ==
My 2016 dare2believe theme is clean, blog-focused, and beautifully designed.  dare2believe theme for WordPress featuring a clean and neat format of posts, each displayed perfectly in their own unique way. I designed it using a mobile-first approach, meaning this flexible layout looks great on any device, big or small and suitable for multiple languages. 

* Mobile-first, Responsive Layout
* Custom Colors
* Custom Header
* The GPL v2.0 or later license.


== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in dare2believe in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
